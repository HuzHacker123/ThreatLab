🚨 ThreatLab — Cybersecurity Learning Platform

An interactive, hands-on cybersecurity platform designed to help students and beginners learn, test, and understand security concepts in a safe environment.

⸻

🌐 Overview

ThreatLab is a practical cybersecurity web application that provides real-world inspired tools and simulations to help users explore security concepts like password strength, encryption, phishing, and network scanning.

Instead of just reading theory, users can interact, experiment, and learn by doing — which is one of the most effective ways to understand cybersecurity.  ￼

⸻

⚡ Features

* 🔐 Password Strength Analyzer
    Evaluate how secure a password is and learn how to improve it.
* 🔑 Cipher Tools (Encryption / Decryption)
    Understand basic cryptography using classic techniques like Caesar Cipher.
* 🎣 Phishing Simulator
    Demonstrates how phishing attacks work (for educational awareness only).
* 🌐 Network Scanner Dashboard
    Analyze basic network information and understand scanning concepts.
* 🦠 Malware File Scanner (Demo)
    Simulate file scanning and learn how malware detection works.
* 💉 SQL Injection Demonstration
    Visualize how insecure queries can be exploited.

⸻

🧠 Purpose

The goal of ThreatLab is to:

* Make cybersecurity easy to understand
* Provide safe, controlled simulations
* Help beginners move from theory → practical skills
* Build awareness of real-world cyber threats
